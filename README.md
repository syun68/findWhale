# FindWhale:whale2:
:::note info
海洋生物の写真投稿サイトです。  
野生の海洋生物情報を位置情報付きで共有できます。  
レスポンシブ対応しておりますので、スマホ・タブレットでもご覧いただけます。
:::

## 使用技術
* Ruby 3.1.2
* Ruby on Rails 7.0.4
* MySQL2 0.5.4
* AWS
* Docker/Docker-compose
* Circle CI/CL
* RSpec
* Google Maps API
